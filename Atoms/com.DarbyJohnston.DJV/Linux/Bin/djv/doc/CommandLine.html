User Interface

-reset_prefsReset the preferences.

OpenGL

-render_filter (zoom out) (zoom in)Set the render filter: Nearest, Linear, Box, Triangle, Bell, B-Spline, Lanczos3, Cubic, Mitchell. Default = Linear, Nearest.
-render_filter_highSet the render filter to high quality settings (Lanczos3, Mitchell).

File Sequences

-seq_compress (value)Set the file sequence compression: Off, Sparse, Range. Default = Sparse.
-seq_auto (value)Set whether auto file sequencing is enabled: False, True. Default = True.
-seq_max (value)Set the maximum allowed size of file sequences. Default = 50000.
-seq_negative (value)Set whether negative numbers are enabled: False, True. Default = False.

Time

-time_units (value)Set the time units: Timecode, Frames. Default = Frames.
-default_speed (value)Set the default speed: 1, 3, 6, 12, 15, 16, 18, 23.976, 24, 25, 29.97, 30, 50, 59.94, 60, 120. Default = 24.

Miscellaneous

-debug_logPrint debug log messages.
-help, -hShow the command line documentation.
-infoShow information about the application.
-aboutShow legal infomration.
