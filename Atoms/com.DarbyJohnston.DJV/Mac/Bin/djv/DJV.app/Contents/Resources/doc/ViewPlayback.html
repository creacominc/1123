Playback Controls

Icon Menu Item Description Keyboard Shortcut
Forward Start forward playback L
Stop Stop playback K
Reverse Start reverse playback J
Toggle Playback Toggle between playback and stopped Spacebar
Loop / Once Play once and stop at the end
Loop / Repeat Play continuously in a loop
Loop / Ping Pong Play forward and then reverse in a loop
Every Frame Playback every frame even if the playback speed cannot be maintained

In/Out Points

In and out points allow you to playback a portion of the sequence.

Icon Menu Item Description Keyboard Shortcut
In/Out Points Toggle whether in/out points are enabled P
Mark In Point Set the current frame as the in point I
Mark Out Point Set the current frame as the out point Shift I
Reset In Point Reset the in point Shift I
Reset Out Point Reset the out point Shift O
