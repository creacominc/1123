[ColorCorrectorMasked]
 
Allows you to do color correction with a gradient based edge masking effect applied on a panoramic frame.
 
This is perfect for creating a graduated neutral density filter effect, fixing circular falloff vingetting on lenses, and target color corrections in LatLong stitching
 
[AlphaDivideMask]
  Input=Input


[GradientMaskBackground]
  Width Default=3840
  Height Default=1920
  Depth Default=0.0
  Color Section (Everything)
  Common
    Motion Blur Default=0
    Quality Default=2


[ColorCorrectorMask]
  Correction (Everything)
  Ranges (Everything)
  Common
    Blend
    Process Red
    Process Green
    Process Blue


[AlphaMultiplyMask]
  Output=Output
